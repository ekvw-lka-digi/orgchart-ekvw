# A simple organisation chart application for public service of Berlin

The app was written with the aim to provide a simple yet sufficient tool to create organisational charts to export human as well as machine appealing files. 

# Getting Started

Clone this repository, go to the directory `app` by:

`cd app`

Then install the required dependencies with:

`yarn install`


## Development

To runs the app in the development mode run:

`yarn start`


Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.


## Production

To build the appp for production to the `build` folder run:

`yarn build`

It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!
